# PB TD

 It features a unique maze, a challenging difficulty curve, and high replayability.
Players build gems and stones to protect the Gem Castle from 50+ waves of enemies.

## Gameplay

Players defend their Gem Castle by constructing a long winding maze. Creeps follow the maze, giving towers more time to deal damage before the enemies reach the castle.

Waves become increasingly difficult and can include enemy skills such as:

- Magic immunity
- Physical immunity
- Disarm
- Flying
- Evasion
- Refraction
- Blink
- Rush
- Thief
- Permanent invisibility

The game consists of two major phases:

1. Build Phase
2. Attack Phase

## Build Phase

During the build phase, players place 5 gems. After placing them, the player chooses only one gem to keep. The other placed gems become stones.

This creates a balance between:

- Building the maze itself
- Keeping valuable gems in good positions
- Creating gem combinations for stronger towers

Each placed gem is random unless the player has skills that improve the chance of specific gem types or qualities.

### Build Phase Actions

| Action | Description |
|---|---|
| SELECT / Check | Keep only the selected gem. |
| REMOVE / Stone | Remove the selected stone. |
| MERGE / ^ | Replace the selected gem with the same type one level higher. |
| MERGE / ^^ | Replace the selected gem with the same type two levels higher. |
| COMBINE / Tower | Combine the correct gems/towers into a stronger tower with special effects. |
| DOWNGRADE / Gem | Downgrade the gem to a random lower quality for 200 gold. |

If the player gets all 3 required gems for a tower during one build phase, they can instantly build that tower.

## Attack Phase

During the attack phase, creeps pass through five checkpoints in order before reaching the Gem Castle. They take the shortest available path between checkpoints.

Towers or gems that deal the most damage in a single round receive an MVP bonus.

### MVP Bonus

| Effect | Value |
|---|---|
| Tower/gem damage increase | +10% |
| Nearby enemy magic resistance | -10% |
| Maximum MVP awards | 10 |
| Max MVP aura range | 600, about 2 tiles |
| Max MVP aura effect | Nearby towers and gems gain +75% damage |

Players can manually direct towers or stop them to focus specific enemies or control which towers receive MVP bonuses.

## Strategies

- Manage MVP bonuses properly to maximize damage in the late game.
- Balance physical and magical damage.
- Stack different levels of Opal auras for attack speed and Sapphire auras for slow.
- Identical aura levels do not stack. For example, two Opal E2 auras only count as one.
- Unless doing a quest, getting one required gem on the desired cell during a build phase means the player can build the rest for the maze, because the combination can be made onto that gem and the other 4 gems can become stones.
